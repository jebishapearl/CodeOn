Finding Maximum Element in an Array

Problem Statement

Given an array of integers, find the maximum element in the array.

Example 1

Input:
5
1 3 5 7 9

Output:
9

Explanation:
The maximum element in the array is 9. Hence, the output is 9.

Example 2

Input:
4
-5 -1 0 3

Output:
3

Explanation:
The maximum element in the array is 3. Hence, the output is 3.

  • 1 ‚â§ n ‚â§ 10‚Å∂ (number of elements in the array)
  • ‚àí10‚Å∂ ‚â§ array elements ‚â§ 10‚Å∂ (elements of the array)

Starter Code

Copy the template into the IDE

#include 

int main() {
    int n;
    scanf("%d", &n);
    
    int arr[n];
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }
    
    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    
    printf("%d\n", max);
    return 0;
}

Explanation

üìã Problem Summary

You are given an array of integers. Your task is to find the maximum value in the array.

üî¢ Input Format

  • First line: Integer n ‚Äî the number of elements.
  • Second line: n space-separated integers.

üß≠ Step-by-Step Approach

  1. Read the value of n and the array of n integers.
  2. Initialize a variable max to the first element of the array.
  3. Loop through the remaining elements.
  4. If any element is greater than max, update max.
  5. After the loop ends, print max.

üß† Key Observation

Comparing each element once is enough to find the maximum.

‚öôÔ∏è Time and Space Complexity

  • Time Complexity: O(n)
  • Space Complexity: O(1) (excluding array input)

üîç Edge Cases to Consider

  • All elements are the same ‚Üí Any of them is the max.
  • All negative numbers ‚Üí Still works, max will be the least negative.
  • Array of length 1 ‚Üí Return that element itself.
