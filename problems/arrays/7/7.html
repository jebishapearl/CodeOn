Counting Multiples of X in an Array

Problem Statement

Given an array of integers, count how many elements in the array are divisible by a given integer x.

Example 1

Input:
5
2 4 6 8 10
2

Output:
5

Explanation:
All elements (2, 4, 6, 8, 10) are divisible by 2, so the count is 5.

Example 2

Input:
6
3 5 7 9 11 12
3

Output:
3

Explanation:
The elements divisible by 3 are 3, 9, and 12, so the count is 3.

Explanation

üßæ Problem Understanding

You are given an array of integers and an integer x. Your goal is to count how many elements in the array are divisible by x.

An element a[i] is divisible by x if a[i] % x == 0.

üî¢ Input Format

  • The first line contains two integers: n (size of array) and x (the divisor).
  • The second line contains n space-separated integers ‚Äî the elements of the array.

üí° Step-by-Step Approach

  1. Read two integers: n and x.
  2. Read n integers into an array.
  3. Initialize a counter to 0.
  4. Loop through each element and check if it is divisible by x using the condition arr[i] % x == 0.
  5. If true, increment the counter.
  6. Print the final counter value.

üìò Language-Specific Notes

  • C: Use scanf and the modulo operator %.
  • Python: Use input().split() and a simple if num % x == 0 check.
  • Java: Use Scanner and loop with arr[i] % x == 0.

‚öôÔ∏è Time and Space Complexity

  • Time Complexity: O(n) ‚Äî one pass through the array.
  • Space Complexity: O(n) ‚Äî storing all elements in the array.

üîç Edge Cases to Consider

  • All elements are divisible by x ‚Üí count = n.
  • No elements are divisible ‚Üí count = 0.
  • x = 1 ‚Üí All numbers are divisible ‚Üí count = n.
  • Negative numbers should still work because mod works for them too.
