Counting Prime Numbers in an Array

Problem Statement

Given an array of integers, count how many elements in the array are prime numbers.

Example 1

Input:
5
2 3 4 5 10

Output:
3

Explanation:
The prime numbers in the array are 2, 3, and 5. Hence, the output is 3.

Example 2

Input:
6
6 7 8 9 10 11

Output:
2

Explanation:
The prime numbers in the array are 7 and 11. Hence, the output is 2.

Constraints

  • 1 ≤ n ≤ 10⁶ (number of elements in the array)
  • −10⁶ ≤ array elements ≤ 10⁶ (elements of the array)

bool is_prime(int num) {
    if (num <= 1) return false;
    for (int i = 2; i * i <= num; i++) {
        if (num % i == 0) {
            return false;
        }
    }
    return true;
}

int main() {
    int n, prime_count = 0;
    scanf("%d", &n);
    
    int arr[n];
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }
    
    for (int i = 0; i < n; i++) {
        if (is_prime(arr[i])) {
            prime_count++;
        }
    }
    
    printf("%d\n", prime_count);
    return 0;
}

Explanation

🧾 Problem Understanding

You're given an array of integers and need to count how many are prime numbers.

A prime number is an integer greater than 1 that has no positive divisors other than 1 and itself.

🔢 Input Format

  • First line: Integer n — the number of elements in the array.
  • Second line: n space-separated integers — the elements of the array.

💡 Step-by-Step Approach

  1. Read the integer n.
  2. Read n integers into an array.
  3. Initialize a variable prime_count to 0.
  4. For each element, check if it is a prime using a helper function is_prime().
  5. Increment prime_count for each prime number found.
  6. Print prime_count.

🧠 Prime-Checking Logic

The function is_prime(num) checks if a number is prime by testing divisibility from 2 to √num.

bool is_prime(int num) {
    if (num <= 1) return false;
    for (int i = 2; i * i <= num; i++) {
        if (num % i == 0) return false;
    }
    return true;
}

⚙️ Time and Space Complexity

  • Time Complexity: O(n√m), where m is the maximum number in the array.
  • Space Complexity: O(1) if not counting input array, else O(n).

🔍 Edge Cases to Consider

  • Array contains all non-prime numbers → Output: 0
  • Array has duplicate primes (e.g., 2 2 2) → Count all
  • Negative numbers and 0, 1 are not prime → Should be ignored
