Count Even Numbers in an Array

Problem Statement

Given an array of integers, count how many even numbers are present in the array.

Example 1

Input:
5
1 2 3 4 5

Output:
2

Explanation:
The even numbers are 2 and 4, so the count is 2.

Example 2

Input:
4
10 11 12 13

Output:
2

Explanation:
The even numbers are 10 and 12, so the count is 2.

Constraints

  • 1 ≤ n ≤ 10⁶ (number of elements in the array)
  • −10⁶ ≤ array elements ≤ 10⁶ (elements of the array)

Starter Code

Copy the template into the IDE

        #include <stdio.h>
        int main() {
            int n, evenCount = 0;
            scanf("%d", &n);
            
            int arr[n];
            for (int i = 0; i < n; i++) {
                scanf("%d", &arr[i]);
            }
            
            for (int i = 0; i < n; i++) {
                if (arr[i] % 2 == 0) {
                    evenCount++;
                }
            }
            
            printf("%d\\n", evenCount);
            return 0;
        }

                if (arr[i] % 2 == 0) {
                    evenCount++;
                }

            printf("Count of even numbers is: %d\n", evenCount);
            return 0;
        }
Explanation

🧾 Problem Understanding

You are given an array of integers and your task is to count how many of them are even. A number is even if it is divisible by 2 (i.e., number % 2 == 0).

🔢 Input Format

  • The first line contains an integer n — the number of elements in the array.
  • The second line contains n space-separated integers — the elements of the array.

💡 Step-by-Step Approach

  1. Read the number of elements n.
  2. Read the n integers into an array.
  3. Initialize a counter variable evenCount to 0.
  4. Loop through the array.
  5. For each element, check if it’s even using arr[i] % 2 == 0.
  6. If it’s even, increment evenCount.
  7. After the loop ends, print the value of evenCount.

📘 Language-Specific Notes

  • C: Uses a simple for loop with array indexing and condition check.
  • Python: Use a loop or list comprehension with if num % 2 == 0 to count.
  • Java: Loop through an int[] and check arr[i] % 2 == 0.

⚙️ Time and Space Complexity

  • Time Complexity: O(n) — Each element is checked once.
  • Space Complexity: O(n) — For storing the array.

🔍 Edge Cases to Consider

  • All elements are even → output = n.
  • No even elements → output = 0.
  • Negative even numbers → should also be counted.
  • Array of size 1 → edge condition check.
