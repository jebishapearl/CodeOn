Sum of Elements in an Array

Problem Statement

Given an array of integers, calculate the sum of all the elements using an iterative approach.

Example 1

Input:
5
1 2 3 4 5

Output:
15

Explanation:
The sum of the array elements is 1 + 2 + 3 + 4 + 5 = 15.

Example 2

Input:
3
10 20 30

Output:
60

Explanation:
The sum of the array elements is 10 + 20 + 30 = 60.

        int main() {
            int n, sum = 0;
            scanf("%d", &n);
            
            int arr[n];
            for (int i = 0; i < n; i++) {
                scanf("%d", &arr[i]);
            }
            
            for (int i = 0; i < n; i++) {
                sum += arr[i];
            }
            
            printf("%d\\n", sum);
            return 0;
        }

Explanation

To calculate the sum of elements in an array, we use a very straightforward iterative approach that works in all 3 languages‚ÄîC, Python, and Java.

üî∂ Step-by-step logic:

  1. First, we read an integer n, which represents the number of elements in the array.
  2. We then declare an array of size n and read n integers into it.
  3. We initialize a variable (e.g., sum) to 0.
  4. We loop over each element in the array and add it to sum.
  5. Finally, we print the value of sum.

üß† Why this works:

We‚Äôre iterating over all elements once, so the time complexity is O(n), which is optimal for this task. It also works for negative numbers, since summation handles both positive and negative integers.

üí° Edge Cases:

  • When all elements are 0 ‚Üí output is 0.
  • When elements include negative numbers ‚Üí they are simply included in the sum.
  • When n = 1 ‚Üí sum is the single element.

üíª Language-Specific Notes:

  • C: Uses dynamic-length arrays (C99 feature). Make sure your compiler supports it.
  • Python: Typically uses input().split() and map(int, ...) to read input and sum it with a loop or sum().
  • Java: Requires Scanner class for input, array creation, and loop for summing.

This logic applies cleanly across all three languages without major structural differences.
