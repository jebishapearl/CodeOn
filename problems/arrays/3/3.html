Sum of Odd Numbers in an Array

Problem Statement

Given an array of integers, calculate the sum of all the odd numbers using an iterative approach.

Example 1

Input:
5
1 2 3 4 5

Output:
9

Explanation:
The odd numbers are 1, 3, and 5, and their sum is 1 + 3 + 5 = 9.

Example 2

Input:
4
10 11 12 13

Output:
24

Explanation:
The odd numbers are 11 and 13, and their sum is 11 + 13 = 24.

Constraints

  • 1 ≤ n ≤ 10⁶ (number of elements in the array)
  • −10⁶ ≤ array elements ≤ 10⁶ (elements of the array)

Starter Code

Copy the template into the IDE

        #include <stdio.h>
        int main() {
            int n, sum = 0;
            scanf("%d", &n);
            
            int arr[n];
            for (int i = 0; i < n; i++) {
                scanf("%d", &arr[i]);
            }
            
            for (int i = 0; i < n; i++) {
                if (arr[i] % 2 != 0) {
                    sum += arr[i];
                }
            }
            
            printf("%d\\n", sum);
            return 0;
        }

Explanation

🔍 Problem Understanding

The goal is to compute the sum of all odd numbers in a given array. A number is odd if it leaves a remainder of 1 when divided by 2. (i.e., num % 2 != 0)

🧾 Input Format

  • The first line contains a single integer n, the number of elements in the array.
  • The second line contains n space-separated integers.

💡 Step-by-Step Approach

  1. Read the number of elements n.
  2. Read n integers into an array.
  3. Initialize a variable sum to 0.
  4. Iterate through the array using a loop.
  5. For each element, check if it's odd using element % 2 != 0.
  6. If it is, add it to sum.
  7. After the loop ends, print the value of sum.

⚙️ Time and Space Complexity

  • Time Complexity: O(n) — Every element is checked exactly once.
  • Space Complexity: O(n) for storing the array.

🔍 Edge Cases to Consider

  • No odd numbers in the array — output should be 0.
  • All elements are odd — sum is total of all elements.
  • Negative odd numbers are valid and should be included in the sum.

📘 Language-Specific Notes

  • C: Uses a variable-length array (VLA) which is allowed in C99 and later. The modulo operator is used directly in the loop. Array input via scanf().
  • Python: Input usually handled using map(int, input().split()). Odd check is done with if num % 2 != 0.
  • Java: Uses Scanner for input, an integer array, and a for loop to conditionally sum odd numbers.
