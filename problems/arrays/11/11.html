Reverse an Array

Problem Statement

Given an array of integers, reverse the array.

Example

Input:

        5
        1 3 7 2 9

Output:

        9 2 7 3 1

Explanation:

The array is reversed from [1, 3, 7, 2, 9] to [9, 2, 7, 3, 1].

Starter Code

Copy the template into the IDE

#include 

int main() {
    int n;
    scanf("%d", &n);

    int arr[n];
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    // Reversing the array
    for (int i = 0; i < n / 2; i++) {
        int temp = arr[i];
        arr[i] = arr[n - i - 1];
        arr[n - i - 1] = temp;
    }

    // Output the reversed array
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}
