Factorial of a Number

Problem Statement

Given a number N, calculate the factorial of N using an iterative approach.

Example 1

Input:
5

Output:
120

Explanation:
The factorial of 5 is calculated as: 5! = 5 √ó 4 √ó 3 √ó 2 √ó 1 = 120.

Example 2

Input:
3

Output:
6

Explanation:
The factorial of 3 is calculated as: 3! = 3 √ó 2 √ó 1 = 6.

View Explanation

The goal of this program is to calculate the factorial of a number N using an iterative approach. The factorial of a number (denoted as N!) is the product of all positive integers from 1 to N. For example, 5! = 5 √ó 4 √ó 3 √ó 2 √ó 1 = 120.

Here's how the logic works:

  • Start by reading the input number N from the user.
  • Initialize a variable (e.g., fact) to 1. This will hold the result of the factorial.
  • Use a loop that runs from 1 to N. In each iteration, multiply fact by the current loop index.
  • After the loop ends, fact will contain the final factorial value, which is then printed.

This logic is the same across all three languages (C, Python, Java). The only differences are:

  • C: Uses scanf and printf for input/output.
  • Python: Uses input() and print().
  • Java: Uses Scanner for input and System.out.println() for output.

This method is efficient and works well for small values of N (as per the constraint 0 ‚â§ N ‚â§ 20).
