Right-Angled Star Pattern

Problem Statement

Given a number N, print a right-angled triangle pattern of stars (*) with N rows using an iterative approach.

Input Format

  • A single integer N (1 ≤ N ≤ 100).

Output Format

  • N lines of output, where the ith line contains i stars (*).

Example

Input:

5

Output:

    *
    **
    ***
    ****
    *****

Starter Code

Copy the template into the IDE

        #include <stdio.h>
        int main() {
            int N;
            printf("enter a number: ");
            scanf("%d", &N);
            for (int i = 1; i <= N; i++) {
                for (int j = 1; j <= i; j++) {
                    printf("*");
                }
                printf("\n");
            }
            return 0;
        }

Hints

Hint 1 🔍

Think about how many rows and columns you need to print in a right-angled triangle pattern.

Hint 2 🤔

Use a loop to print rows, and inside each row, use another loop to print stars.

Hint 3 🛠️

The number of stars in each row is equal to the row number.

Hint 4 🔄

We used a similar approach in the Sum of first N numbers problem, where we iterated from 1 to N. Here, we do the same but print stars instead of summing numbers.

Try It Yourself

Solution

View Solution 
#include <stdio.h>

int main() {
    int N, sum = 0;
    printf("enter a number: ");
    scanf("%d", &N);
    for (int i = 1; i <= N; i++) {
        sum += i;
    }
    printf("%d\n", sum);
    return 0;
}
View Explanation

1 · What pattern are we printing?

A left‑aligned right‑angled triangle where the i‑th row contains exactly i asterisks: 

*
**
***
****
...

For an input N, we therefore need N rows, and each row’s star‑count is its own row‑number.

2 · Core idea (language‑agnostic)

  • Loop from i = 1 to N (one row per i).
  • During each iteration, output i stars, then move to the next line.

3 · How C and Java implement it — nested loops


// outer loop: rows
for (int i = 1; i <= N; i++) {

    // inner loop: print 'i' stars on this row
    for (int j = 1; j <= i; j++) {
        printf("*");              // C
        // System.out.print("*"); // Java
    }

    printf("\n");                  // C
    // System.out.println();       // Java
}

The outer loop decides how many rows to print.
The inner loop prints exactly i stars for that row.
After finishing the inner loop, a newline starts the next row.

4 · How Python implements it — string multiplication trick

N = int(input())          # read N
for i in range(1, N + 1):        # single loop = rows
    print('*' * i)               # '*' multiplied i times

Python’s strings can be “multiplied” by an integer, so '*' * i instantly produces a string containing i asterisks. That removes the need for an inner loop and makes the code shorter. Functionally, it’s the same: one print per row, newline added automatically by print().

5 · Complexities

  • Time Complexity: O(N²)  — we print 1 + 2 + … + N ≈ N(N+1)/2 stars.
  • Space Complexity: O(1)  — we only store loop counters (Python’s temporary string is discarded each iteration).

6 · Take‑away differences

  • C / Java: Two explicit loops, very clear step‑by‑step control.
  • Python: One loop plus string multiplication  →  more concise but relies on a language feature.
