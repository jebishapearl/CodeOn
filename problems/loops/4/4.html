Sum of First N Odd Numbers

Problem Statement

Given a number N, calculate the sum of the first N odd numbers using an iterative approach.

Example 1

Input:
4

Output:
16

Explanation:
The first 4 odd numbers are: 1, 3, 5, 7. The sum of these numbers is: 1 + 3 + 5 + 7 = 16.

Example 2

Input:
3

Output:
9

Explanation:
The first 3 odd numbers are: 1, 3, 5. The sum of these numbers is: 1 + 3 + 5 = 9.

#include 

int main() {
    int N, sum = 0;
    printf("enter a number: ");
    scanf("%d", &N);
    for (int i = 1; i <= 2*N-1; i += 2) {
        sum += i;
    }
    printf("%d\n", sum);
    return 0;
}
View Explanation

The goal is to calculate the sum of the first N odd numbers. Odd numbers follow the pattern: 1, 3, 5, 7, 9, ...

The logic used in the code:

  • Initialize a variable sum to 0.
  • Start a for loop from 1, which is the first odd number.
  • Increment the loop variable by 2 in each iteration to get the next odd number.
  • Run the loop until i reaches 2*N - 1. This ensures you get exactly N odd numbers.
  • In each iteration, add the current odd number to sum.
  • After the loop ends, sum will hold the total of the first N odd numbers.

The same logic applies across C, Python, and Java ‚Äî only the syntax for input/output differs.
