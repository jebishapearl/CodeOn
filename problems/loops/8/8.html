Multiplication Table (1 to 10)

Problem Statement

Given a number N, print its multiplication table from 1 to 10 using a loop.

Example

5

        5 √ó 2 = 10
        5 √ó 3 = 15

        5 √ó 5 = 25
        5 √ó 6 = 30

        5 √ó 8 = 40

Explanation

For the input number 5, the multiplication table is created by multiplying 5 with each number from 1 to 10. The results are displayed in the format N √ó i = result, where N is the input number and i is the multiplier ranging from 1 to 10. In this case, the output shows the results of multiplying 5 by each number from 1 to 10.

Try It Yourself

Solution

View Solution 
#include <stdio.h>

int main() {
    int N;
    printf("enter a number: ");
    scanf("%d", &N);
    for (int i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", N, i, N * i);
    }
    return 0;
}
