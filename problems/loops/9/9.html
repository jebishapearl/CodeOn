Multiplication Table (10 to 1)

Problem Statement

Given a number N, print its multiplication table from 10 down to 1.

Example

Input

5

Output

            5 √ó 10 = 50
            5 √ó 9 = 45
            5 √ó 8 = 40
            5 √ó 7 = 35
            5 √ó 6 = 30
            5 √ó 5 = 25
            5 √ó 4 = 20
            5 √ó 3 = 15
            5 √ó 2 = 10
            5 √ó 1 = 5

Explanation

For the input number 5, the multiplication table is created by multiplying 5 with each number starting from 10 down to 1. The results are displayed in the format N √ó i = result, where N is the input number and i is the multiplier, starting from 10 and decreasing by 1 with each iteration. In this case, the output shows the results of multiplying 5 by each number from 10 down to 1.

Try It Yourself

Solution

View Solution 
#include <stdio.h>

int main() {
    int N;
    printf("enter a number: ");
    scanf("%d", &N);
    for (int i = 10; i >= 1; i--) {
        printf("    %d √ó %d = %d\n", N, i, N * i);
    }
    return 0;
}

Common Logic: All three languages perform the same task:

  • Take an integer input N
  • Use a loop that starts at 10 and goes down to 1
  • Print N √ó i = result on each line

Language-Specific Notes:

  • C: Uses scanf to read input and printf for formatted output with format specifiers like %d.
  • Python: Uses input() for reading input and f-strings (or format()) to format the output cleanly. Loop uses range(10, 0, -1) to count backwards.
  • Java: Uses Scanner to read input and System.out.println to display results. Loop runs in reverse using i--.

Though syntax varies slightly, the logic is identical in all three. Python remains the most concise, while C and Java require more boilerplate code.
