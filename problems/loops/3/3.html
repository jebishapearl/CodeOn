Solution

#include <stdio.h>

int main() {
    int N, sum = 0;
    printf("enter a number: ");
    scanf("%d", &N);
    for (int i = 2; i <= N; i += 2) {
        sum += i;
    }
    printf("%d\n", sum);
    return 0;
}
View Explanation

The goal is to calculate the sum of all even numbers between 1 and N (inclusive). An even number is any number divisible by 2 (like 2, 4, 6, ...).

The logic used in the code:

  • Start by initializing a variable sum to 0.
  • Use a for loop starting from 2 (the first even number) and increment by 2 each time (i.e., i += 2).
  • Run the loop as long as i ‚â§ N. For each even number i, add it to sum.
  • After the loop ends, sum will contain the total of all even numbers from 1 to N.

This logic is the same in C, Python, and Java. Only the syntax for input and output differs between the languages.
