Sum of N Natural Numbers

Problem Statement

Given a number N, calculate the sum of the first N natural numbers using an iterative approach.

Example 1

Input:
5

Output:
15

Explanation:
The first 5 natural numbers are: 1, 2, 3, 4, 5. The sum of these numbers is: 1 + 2 + 3 + 4 + 5 = 15.

Example 2

Input:
3

Output:
6

Explanation:
The first 3 natural numbers are: 1, 2, 3. The sum of these numbers is: 1 + 2 + 3 = 6.
