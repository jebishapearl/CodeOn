Armstrong Number

Problem Statement

Given a number N, determine whether it is an Armstrong number. An Armstrong number (also known as a narcissistic number) is a number that is equal to the sum of its own digits each raised to the power of the number of digits.

Example 1

Input:
153

Output:
YES

Explanation:
153 has 3 digits. 1³ + 5³ + 3³ = 1 + 125 + 27 = 153, so it is an Armstrong number.

Example 2

Input:
123

Output:
NO

Explanation:
123 has 3 digits. 1³ + 2³ + 3³ = 1 + 8 + 27 = 36, which is not equal to 123, so it is not an Armstrong number.

Constraints

  • 1 ≤ N ≤ 10⁶

Solution

View Solution 
        #include <stdio.h>
        #include <math.h>
        int main() {
            int N, sum = 0, temp, remainder, digits = 0;
            printf("enter a number: ");
            scanf("%d", &N);
            temp = N;
        
            // Count the number of digits in N
            while (temp != 0) {
                temp /= 10;
                digits++;
            }
        
            temp = N;
            // Calculate the sum of each digit raised to the power of 'digits'
            while (temp != 0) {
                remainder = temp % 10;
                sum += pow(remainder, digits);
                temp /= 10;
            }
        
            // Check if the sum equals the original number
            if (sum == N)
                printf("YES\n");
            else
                printf("NO\n");
        
            return 0;
        }

Explanation

View Explanation

An Armstrong number is a number that is equal to the sum of its digits each raised to the power of the number of digits.

Steps to solve:

  1. Count the digits: First, we find out how many digits are in the number N. This is usually done by dividing N by 10 repeatedly until it becomes 0.
  2. Break the number down: We extract each digit using % 10, just like in the “Reverse a Number” or “Count Digits” problems.
  3. Raise to power: For each digit, raise it to the power of the digit count using pow() in C and Java, or ** in Python.
  4. Compare: Sum all powered digits. If this sum equals the original number, then it's an Armstrong number.

Does the approach change in Python or Java?

Not really. The steps remain the same:

  • Use ** in Python for power, pow() in C and Java.
  • In Python, we use len(str(N)) to count digits easily instead of a loop.
  • Syntax and variable declaration rules differ slightly, but the core logic is the same.

Example: 153

  • 153 has 3 digits.
  • Digits: 1, 5, 3 → Raised to power 3: 1³, 5³, 3³ → 1 + 125 + 27 = 153
  • Since the sum equals the number, output is YES.

Another Example: 123

  • Digits: 1, 2, 3 → 1³ + 2³ + 3³ = 1 + 8 + 27 = 36 ≠ 123
  • So the output is NO.
