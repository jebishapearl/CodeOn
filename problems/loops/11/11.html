Solution

#include <stdio.h>

int main() {
    int N, count = 0;
    printf("enter a number: ");
    scanf("%d", &N);

    // Special case for 0
    if (N == 0) {
        count = 1;
    } else {
        while (N > 0) {
            count++;
            N /= 10;
        }
    }

    printf("%d\n", count);
    return 0;
}

Explanation

View Explanation

To count the number of digits in a number, we repeatedly divide it by 10. Each division removes the last digit, and we increase a counter to keep track of how many digits we removed. When the number becomes 0, we stop.

Approach

  • Initialize a variable count to 0.
  • While the number N is greater than 0:
    • Increment count
    • Divide N by 10
  • If N is 0 initially, we manually set count = 1 (since 0 has one digit).

Example

If the number is 12345:

  1. 12345 ‚Üí count = 1
  2. 1234 ‚Üí count = 2
  3. 123 ‚Üí count = 3
  4. 12 ‚Üí count = 4
  5. 1 ‚Üí count = 5
  6. 0 ‚Üí loop ends

Final count = 5

Language Notes

  • C, Python, and Java all use the same approach: loop + division + counter.
  • Python and Java automatically handle large numbers. In C, make sure to use long if testing large values (up to 10‚Åπ).
