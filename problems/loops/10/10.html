Reverse a Number

Output:
54321

Explanation:
The number 12345 has its digits reversed to 54321.

#include <stdio.h>

int main() {
    int N, reversed = 0;
    printf("enter a number: ");
    scanf("%d", &N);
    
    while (N > 0) {
        reversed = reversed * 10 + (N % 10);
        N /= 10;
    }
    
    printf("%d\\n", reversed);
    return 0;
}

Explanation

View Explanation

This problem asks us to reverse the digits of a number. For example, if the number is 12345, we need to transform it into 54321.

Approach

  • We extract the last digit of the number using the modulus operator (%).
  • We build the reversed number by multiplying the current reversed value by 10 and adding the extracted digit.
  • We remove the last digit from the original number using integer division (/ in C/Java, // in Python).
  • This process continues until the number becomes 0.

Example Walkthrough

Let‚Äôs say N = 123. Here's how the process works step by step:

  1. Initial reversed = 0, N = 123
  2. Take last digit ‚Üí 3. Now reversed = 0 * 10 + 3 = 3, N = 12
  3. Take last digit ‚Üí 2. Now reversed = 3 * 10 + 2 = 32, N = 1
  4. Take last digit ‚Üí 1. Now reversed = 32 * 10 + 1 = 321, N = 0

Final output is 321.

Language Notes

  • C, Python, and Java all use the same approach.
  • All three use a loop, the modulo operator (%), and integer division to peel off digits and build the reversed number.
